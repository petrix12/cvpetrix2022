Desarrollo de un programa modular para el ajuste de datos experimentales con modelos no lineales (Tesis de Grado - Ingeniería Química)
https://saber.ucv.ve/handle/10872/71
junio 2011
Software desarrollado para la resolución de problemas complejos de optimización y ajuste de parámetros en modelos matemáticos no lineales. La aplicación implementa múltiples algoritmos numéricos (Gauss-Newton, Levenberg-Marquardt, Simplex Modificado, entre otros) para la interpretación de datos experimentales en Ingeniería Química.

Este proyecto demuestra mi capacidad para traducir fundamentos matemáticos y algoritmos científicos en soluciones de software funcionales y modulares, sentando las bases de mi carrera en el desarrollo de sistemas de alta precisión.
